# NIHONGO - 日语学习网页应用

这是一个基于 Web 技术 (HTML, CSS, JavaScript) 构建的日语学习 Progressive Web App (PWA)。

## 📁 项目位置
当前项目位于您的电脑： `e:\Antigravity\`

## 🚀 如何运行
1.  **直接打开**：双击 `index.html` 文件即可在浏览器中运行（部分功能可能受浏览器安全策略限制）。
2.  **本地服务器 (推荐)**：使用 VS Code 的 "Live Server" 插件或任何静态文件服务器运行，效果最佳。
3.  **从手机访问**：
    *   确保手机和电脑连在同一个 WiFi 下。
    *   在电脑上开启 HTTP 服务器（例如 `python -m http.server 8000` 或 Live Server）。
    *   手机浏览器输入电脑的局域网 IP 地址（例如 `http://192.168.1.x:8000`）。

## 💾 如何备份
为了防止文件丢失，您可以采取以下任一措施：

1.  **复制文件夹**：直接将整个 `Antigravity` 文件夹复制到 U 盘或移动硬盘。
2.  **压缩存档**：右键点击文件夹 -> "发送到" -> "压缩(zipped)文件夹"，将生成的压缩包发送到微信/QQ或保存到网盘。
3.  **使用 Git (高级)**：如果您熟悉 Git，可以初始化仓库并推送到 GitHub/Gitee。

## 📱 安卓兼容性说明
本项目已针对安卓设备进行了优化：
*   **音频**：使用有道词典在线 API，无需安装语音包。
*   **显示**：修复了部分安卓浏览器的渲染兼性问题。
